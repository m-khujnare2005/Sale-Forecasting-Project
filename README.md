# Sale-Forecasting-Project

## Overview

This project predicts future sales using historical Walmart sales data and the Prophet time-series forecasting model.

## Technologies Used

* Python
* Google Colab
* Pandas
* Matplotlib
* Prophet

## Features

* Historical Sales Analysis
* Trend Analysis
* Seasonal Analysis
* Holiday Impact Analysis
* Future Sales Forecasting
* Forecast vs Actual Visualization

## Dataset

Walmart Sales Dataset

## Model

Prophet Time Series Forecasting

## Result

Successfully forecasted future sales and analyzed trends, seasonality, and holiday effects to support business decision-making.
